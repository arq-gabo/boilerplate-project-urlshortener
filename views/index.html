URL Shortener

Example

https://www.google.com - { original_url: https://www.google.com , short_url: 1 } - Valid URL
http://www.facebook.com - { original_url: http://www.facebook.com , short_url: 2 } - Valid URL
www.google.com - {"error": "Invalid URL"}
https://www.thisWebSiteNotExist.com - {"error": "Invalid Hostname"}
